# USAGE

## Fetch dependencies
```
$ yarn install
```

## Markdown to HTML with [Pandoc](https://pandoc.org)

```
$ pandoc --standalone -c style.css -o resume.html resume.md
```

## HTML to PDF with [WKhtmlToPDF](https://wkhtmltopdf.org)

```
$ wkhtmltopdf -L 20mm -R 20mm resume.html resume.pdf
```
